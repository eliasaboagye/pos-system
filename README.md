# POS System

Phase 0 scaffold for a Point of Sale (POS) system built with Node.js, Express.js, MySQL, Sequelize ORM, and a plain HTML/CSS/JavaScript frontend.

## Structure

- `backend/` contains the API server, database configuration, and backend modules.
- `frontend/` contains the static pages and browser-side JavaScript files.

## Getting Started

1. Create the backend environment variables in `backend/.env`.
2. Install backend dependencies.
3. Start the backend server with `node server.js` from the `backend/` directory.
